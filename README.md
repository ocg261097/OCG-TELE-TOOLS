### Overview
OCG TELE TOOLS is a powerful tool for managing and converting Telegram data files. This tool supports various operations to facilitate seamless interactions with Telegram data, including session management, file conversions, and more.

### What are TGNET, TDATA, and Session files?

- **TGNET**: TGNET files are used by Telegram to store account-related data such as login credentials and session information in an encrypted format. These files help Telegram users remain logged in across different devices without requiring reauthentication.
  
- **TDATA**: TDATA is a format used by older versions of Telegram to store user sessions and login data. It contains sensitive information about a user’s account and can be used to maintain login status on different Telegram instances.

- **Session**: A session file stores user credentials and login information to keep a user logged in without having to re-enter credentials each time. Session files are specific to the device and contain encrypted data to ensure user security.
